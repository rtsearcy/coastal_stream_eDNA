Data and Code for the Coastal Stream eDNA Project

Data and Python Code for the Scott Creek eDNA study conducted in 2019 and 2020. Collaboration between the Monterey Bay Aquarium Research Institute (MBARI), the Monterey Bay Aquarium, Stanford University, and the National Atmospheric and Oceanic Administration (NOAA).

Datasets in the 'data' folder include Environmental Sample Processor (ESP) logs, qPCR output and standards, eDNA concentrations, environmental data, and fish trap counts. 

Code in the 'code' folder used to aggregate and analyze data for the manuscript.

Figures and tables are also provided.
